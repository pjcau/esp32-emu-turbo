# Test ROMs — Homebrew / Public Domain

These ROM files are **freely distributable homebrew games** used for testing Retro-Go on the ESP32 Emu Turbo. No copyrighted commercial ROMs are included.

| System | ROM | Author | License |
|--------|-----|--------|---------|
| NES | Owlia | Gradual Games | Freeware |
| GB | Blargg's CPU Instructions | Blargg | Public domain |
| GBC | ucity v1.3 | AntonioND | GPL-3.0 |
| SMS | Silver Valley | Enrique Ruiz | Freeware |
| GG | Swabby v1.11 | Anders S. Jensen | Freeware |
| PCE | Reflectron | Aetherbyte | Freeware |
| GEN | Miniplanets | Sik | Freeware |
| SNES | Super Boss Gaiden v1.2 | Dieter Von Laser | Freeware |

## Sources

- [retrobrews](https://github.com/retrobrews) — curated homebrew ROM collections
- [AntonioND/ucity](https://github.com/AntonioND/ucity) — GBC city builder
- [SNES-SuperBossGaiden](https://github.com/snesdev0815/SNES-SuperBossGaiden) — SNES platformer
- [EmuDeck/emudeck-homebrew](https://github.com/EmuDeck/emudeck-homebrew) — Game Gear homebrew
- [retrio/gb-test-roms](https://github.com/retrio/gb-test-roms) — Blargg's test ROMs

## Usage

Copy to SD card using the setup script:

```bash
sudo ./scripts/setup-sdcard.sh /dev/sdX
```
